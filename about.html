আমাদের সম্পর্কে

Easy Coding Space হলো বাংলা ভাষায় ওয়েব ডেভেলপমেন্ট শেখার একটি বিনামূল্যের শিক্ষামূলক প্ল্যাটফর্ম।

আমাদের লক্ষ্য

  • সহজ বাংলা ভাষায় HTML, CSS ও JavaScript শেখানো
  • বাস্তব উদাহরণ ও প্রজেক্টের মাধ্যমে হাতে-কলমে শিক্ষা
  • সম্পূর্ণ বিনামূল্যে মানসম্পন্ন কোডিং রিসোর্স প্রদান
  • বাংলাভাষী ডেভেলপারদের একটি শক্তিশালী কমিউনিটি গড়ে তোলা

আমরা কী শেখাই

  • HTML — ওয়েবসাইট তৈরির মূল ভাষা
  • CSS — ওয়েব পেজ ডিজাইন করার ভাষা
  • JavaScript — ওয়েব পেজকে ইন্টারেক্টিভ করার ভাষা
  • Git ও GitHub — কোড ম্যানেজমেন্ট ও ভার্সন কন্ট্রোল
  • Tailwind CSS — আধুনিক CSS ফ্রেমওয়ার্ক
  • NPM — JavaScript প্যাকেজ ম্যানেজার
  • Terminal — কমান্ড লাইন ব্যবহার

নির্মাতার কথা

আমি ফরীদুর রহমান — Easy Coding Space-এর প্রতিষ্ঠাতা।

আমি লক্ষ্য করেছি যে বাংলা ভাষায় প্রোগ্রামিং শেখার মানসম্পন্ন কোনো সাইট নেই। জনপ্রিয় সব প্রোগ্রামিং টিউটোরিয়াল সাইট ইংরেজি বা অন্য ভাষায়। যারা ইংরেজি ভালো জানেন না তাদের জন্য প্রোগ্রামিং শিখতে অনেক বেগ পোহাতে হয়।

প্রোগ্রামিং বা ওয়েব ডেভেলপমেন্ট এমনিতেই কঠিন — তার উপর ভিন্ন একটা ভাষায় শিখতে হলে কনসেপ্টগুলো পরিপূর্ণভাবে আত্মস্থ করা প্রায় অসম্ভব হয়ে পড়ে। তাই আমি এই সাইট বানিয়েছি — যাতে যেকেউ নিজের মাতৃভাষায় সহজে প্রোগ্রামিং শিখতে পারে।
