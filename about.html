আমাদের সম্পর্কে

Easy Coding Space হলো বাংলা ভাষায় ওয়েব ডেভেলপমেন্ট শেখার একটি বিনামূল্যের শিক্ষামূলক প্ল্যাটফর্ম। আমাদের লক্ষ্য — বাংলাদেশ ও সারা বিশ্বের বাংলাভাষী মানুষদের কাছে কোডিং শেখাকে সহজ ও সুলভ করে তোলা।

আমাদের লক্ষ্য

  • সহজ বাংলা ভাষায় HTML, CSS ও JavaScript শেখানো
  • বাস্তব উদাহরণ ও প্রজেক্টের মাধ্যমে হাতে-কলমে শিক্ষা
  • সম্পূর্ণ বিনামূল্যে মানসম্পন্ন কোডিং রিসোর্স প্রদান
  • বাংলাভাষী ডেভেলপারদের একটি শক্তিশালী কমিউনিটি গড়ে তোলা

আমরা কী শেখাই

Easy Coding Space-এ আপনি শিখতে পারবেন:

  • HTML — ওয়েবসাইট তৈরির মূল ভাষা
  • CSS — ওয়েব পেজ ডিজাইন করার ভাষা
  • JavaScript — ওয়েব পেজকে ইন্টারেক্টিভ করার ভাষা
  • Git ও GitHub — কোড ম্যানেজমেন্ট ও ভার্সন কন্ট্রোল
  • Tailwind CSS — আধুনিক CSS ফ্রেমওয়ার্ক
  • NPM — JavaScript প্যাকেজ ম্যানেজার
  • Terminal — কমান্ড লাইন ব্যবহার

নির্মাতার কথা

ফরীদুর রহমান — Easy Coding Space-এর প্রতিষ্ঠাতা ও কন্টেন্ট নির্মাতা। তিনি বিশ্বাস করেন যে ভাষা কখনো শেখার পথে বাধা হওয়া উচিত নয়। তাই বাংলায় কোডিং শেখার এই প্ল্যাটফর্ম তৈরি করেছেন যাতে যেকেউ নিজের ভাষায় প্রোগ্রামিং শিখতে পারে।
