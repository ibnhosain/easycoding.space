কোড লেখা শেখো

বাংলা ভাষায় কোডিং শেখো বিশ্বের সবচেয়ে বড় ওয়েব ডেভেলপার সাইটের সাথে।

এখনই শুরু হোক জার্নিটা!

HTML

ওয়েবসাইট তৈরির মূল ভাষা

পড়ে পড়ে শেখা ভিডিও দেখে শেখা গুরুত্বপূর্ণ ট্যাগ লিস্ট

HTML Example:

<!DOCTYPE html>
<html>
<head>
<title>HTML Tutorial</title>
</head>
<body>

<h1>This is a heading</h1>
<p>This is a paragraph.</p>

</body>
</html>

CSS

ওয়েব পেজের ডিজাইন বা রূপ দেওয়ার ভাষা

পড়ে পড়ে শেখা ভিডিও দেখে শেখা গুরুত্বপূর্ণ প্রপার্টি লিস্ট

CSS Example:

body {
  background-color: lightblue;
}

h1 {
  color: white;
  text-align: center;
}

p {
  font-family: verdana;
}

JavaScript

ওয়েব পেজের কার্যক্ষমতা যোগ করার ভাষা

পড়ে পড়ে শেখা ভিডিও দেখে শেখা সম্পূর্ণ JS রেফারেন্স

JavaScript Example:

<button onclick="myFunction()">Click Me!</button>

<script>
function myFunction() {
  let x = document.getElementById("demo");
  x.style.fontSize = "25px";
  x.style.color = "red";
}
</script>

Git

কোড ম্যানেজমেন্ট ও ভার্সন কন্ট্রোল সিস্টেম

পড়ে পড়ে শেখা ভিডিও দেখে শেখা গুরুত্বপূর্ণ কমান্ড লিস্ট

Git Example:

# নতুন repository শুরু করো
git init

# ফাইল যোগ করো
git add .

# পরিবর্তন সংরক্ষণ করো
git commit -m "first commit"

# GitHub-এ পাঠাও
git push origin main

NPM

JavaScript প্যাকেজ ম্যানেজার

পড়ে পড়ে শেখা ভিডিও দেখে শেখা গুরুত্বপূর্ণ কমান্ড

NPM Example:

# প্রজেক্ট শুরু করো
npm init -y

# প্যাকেজ ইন্সটল করো
npm install express

# স্ক্রিপ্ট চালাও
npm run start

# প্যাকেজ আপডেট করো
npm update

Tailwind

আধুনিক Utility-First CSS ফ্রেমওয়ার্ক

পড়ে পড়ে শেখা ভিডিও দেখে শেখা Class রেফারেন্স

Tailwind Example:

<div class="flex items-center
     justify-center
     bg-blue-500
     text-white
     p-4 rounded-xl
     shadow-lg">
  Hello Tailwind!
</div>

Terminal

Command Line ও Bash শেখো

পড়ে পড়ে শেখা ভিডিও দেখে শেখা কমান্ড রেফারেন্স

Terminal Example:

# বর্তমান ফোল্ডার দেখো
$ ls -la

# নতুন ফোল্ডার বানাও
$ mkdir my-project

# ফোল্ডারে ঢুকো
$ cd my-project

# ফাইল তৈরি করো
$ touch index.html
