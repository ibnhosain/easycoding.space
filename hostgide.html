🚀 GitHub Pages দিয়ে Full Website Launch Guide (Complete)

এই সেকশনে তুমি শিখবে—কোড আপলোড থেকে শুরু করে Domain, SEO, HTTPS, Google Search পর্যন্ত সবকিছু একসাথে। এটা ফলো করলে আলাদা কিছু খুঁজতে হবে না।

🔷 শুরু করার আগে যা লাগবে

  • GitHub Account
  • VS Code বা যেকোনো Code Editor
  • Git Install (optional কিন্তু recommended)
  • HTML, CSS, JS ফাইল
  • Responsive Design
  • Domain (optional)
✅ Git + GitHub জানা থাকলে ভবিষ্যতে সব ওয়েব প্রজেক্ট সহজ হয়ে যাবে।

🔷 GitHub Setup

Repository তৈরি করো

1. GitHub → New Repository 2. নাম দাও 3. Public রাখো 4. index.html আপলোড করো

Pages চালু করো

Settings → Pages Branch: main Save → Done

🔷 Custom Domain সেটআপ (যদি থাকে)

GitHub Custom Domain

Settings → Pages Custom domain → yourdomain.com Save

🔷 HTTPS চালু করা

Settings → Pages → Enforce HTTPS ✔
🔒 HTTPS চালু হলে সাইট Secure হয় এবং Google Ranking ভালো হয়।

🔷 গুরুত্বপূর্ণ ফাইল

  • index.html (Main Page)
  • 404.html (Error Page)
  • favicon.ico (Logo)
  • style.css (Design)
  • script.js (Function)
  • robots.txt (SEO)
  • sitemap.xml (Google Index)

🔷 SEO Setup (Google এ আনার জন্য)

Website Title
robots.txt → Google Crawl Control sitemap.xml → All Pages List

🔷 Google Search Console

1. Add Website 2. Verify Domain 3. Submit Sitemap 4. Wait for Index
🔍 Google এ আসতে ১ দিন থেকে ৭ দিন লাগতে পারে।

🔷 Website Performance

  • Image Compress (WebP use করো)
  • Unused CSS Remove
  • JS Minify
  • Lazy Load Images

🔷 Mobile Friendly Check

📱 Chrome → Inspect → Device Mode
⚠️ Mobile Responsive না হলে Google Ranking কমে যায়।
